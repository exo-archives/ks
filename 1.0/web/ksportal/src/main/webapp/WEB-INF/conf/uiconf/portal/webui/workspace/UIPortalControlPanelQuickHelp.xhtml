eXo Portal Control Panel Quick Help.
  • Help: This quick help
  • Basic Customization: In the basic customization panel, you can make some quick change to the portal such changing the skin, language or create a new page
  • Advanced Customization: In the Advanced Customization panel, you can recustomize the entire portal. This panel is for the advanced user only
  • Toggle Ajax: You can use this button to switch back and forth the ajax mode and the conventional mode. In the ajax mode, you should see the response of the request is much faster since only a partial of the portal is updated in each request.
