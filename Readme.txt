Hi, There are is some new features and changes of KS RC5

New features:
  FAQ
   - Import/export categories and questions
   - Add/move/delete/edit question from faq home
  Forum
   - Import/export categories and forum 

Changes:
  FAQ: 
   - Apply nodetype exo:faqQuestionHome for questions node instead of nt:unstructured nodetype
   - Apply nodetype exo:faqCategoryHome for categories node instead of nt:unstructured nodetype
  Forum
   - Apply nodetype exo:forumHome for ForumService node instead of nt:unstructured nodetype
   - Apply nodetype exo:userProfileHome for UserAdministration node instead of nt:unstructured nodetype
   - Apply nodetype exo:userProfiles for UserProfile node instead of nt:unstructured nodetype
   - Apply rich text editor(FCK) for edit notifying content. Get default notifying content
